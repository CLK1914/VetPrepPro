# VetPrep Pro – FFA Vet Science

VetPrep Pro is an interactive study app for FFA Vet Science students. It combines **flashcards, quizzes, and a virtual pet system** to make studying fun and rewarding.

---

## Features

- **Flashcards**: Study key veterinary terms with images and definitions.
- **Quiz**: Test your knowledge with multiple-choice questions.
- **Virtual Pet**: Earn coins and XP by interacting with your pet. Happiness increases as you play.
- **Light/Dark Theme**: Toggle between themes for a comfortable study experience.
- **Progress Tracking**: Coins and XP accumulate as you study and play.
- **PWA Support**: Can be installed on mobile devices like a native app.

---

## How to Use

1. Open the app in a modern browser: `index.html` or via GitHub Pages.
2. Navigate through **Home**, **Flashcards**, **Quiz**, and **Pet** tabs.
3. Flip flashcards and answer quizzes to earn coins and XP.
4. Play with your virtual pet to increase happiness.
5. On mobile, add the app to your home screen for full PWA experience.

---

## Deployment via GitHub Pages

1. Create a repository on GitHub (e.g., `VetPrepPro`).
2. Upload all files: `index.html`, `style.css`, `script.js`, `manifest.json`, `sw.js`, `icon.png`.
3. Go to **Settings → Pages**.
4. Set **Branch** to `main` (or `master`) and folder to `/ (root)`.
5. Save and visit the live site: `https://CLK1914.github.io/VetPrepPro/`.
