**VetPrep Pro** is an interactive web app designed to help FFA Vet Science students study efficiently while having fun. It combines **flashcards, quizzes, and a virtual pet system** to make learning engaging.

---

## Features

- **Flashcards**: Learn key veterinary terms with images and definitions.  
- **Quiz**: Test your knowledge with multiple-choice questions.  
- **Virtual Pet**: Earn coins and XP by interacting with your pet. Happiness increases as you play.  
- **Light/Dark Theme**: Toggle between themes for comfortable viewing.  
- **Progress Tracking**: Coins and XP accumulate as you study and play.  
- **PWA Support**: Install the app on mobile devices for a native-like experience, including offline support.

---

## How to Use

1. Open the app in a modern browser: `index.html` or via GitHub Pages.  
2. Navigate through the **Home**, **Flashcards**, **Quiz**, and **Pet** tabs.  
3. Flip flashcards and answer quizzes to earn coins and XP.  
4. Play with your virtual pet to increase happiness.  
5. On mobile, add the app to your home screen to use it like a native app.
